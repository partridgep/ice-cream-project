# Ice Cream Selector
A simple way to pick your ice cream based on either flavor or brand.


## Getting Started
[Click Here](https://seir-ice-creams.herokuapp.com/) to get started.

## Technologies Used
* HTML
* CSS
* JavaScript/jQuery
* Node.js
* MongoDB
* Mongoose
* Google Fonts
